# Mastercard Financial Modeling & Valuation (Excel Project)

## Project Overview

This project presents a comprehensive **financial modeling and valuation analysis of Mastercard Inc.** built entirely in **Microsoft Excel**.

The model analyzes the company’s **historical financial performance**, forecasts **future financial statements**, and estimates the **intrinsic value** using multiple valuation techniques.

The objective of this project is to demonstrate **practical skills in financial modeling, corporate valuation, and financial statement analysis**, similar to the work performed by **equity research analysts and investment banking professionals**.

---

## Company Overview

**Mastercard Inc.** is a global technology company in the payments industry that connects **consumers, financial institutions, merchants, governments, and businesses worldwide**.

The company enables **electronic payments and digital transaction processing** and provides a wide range of **payment solutions and financial technology services**.

---

## Project Objectives

* Build a structured **financial model using historical financial data**
* Analyze the company’s **financial performance using key financial ratios**
* Forecast **future financial statements based on financial assumptions**
* Estimate the company’s **intrinsic value using multiple valuation methods**
* Compare the company’s **financial performance with industry peers**

---

## Model Structure

### 1. Assumptions

Contains the key inputs used for forecasting:

* Revenue growth assumptions
* Operating margin assumptions
* Tax rate assumptions
* Capital expenditure assumptions
* Discount rate inputs

---

### 2. Financial Statements

The model includes a complete **three-statement financial model**:

* Income Statement
* Balance Sheet
* Cash Flow Statement

All financial statements are **fully linked and dynamically updated based on assumptions**.

---

### 3. Financial Analysis

Includes detailed financial performance analysis:

* Common Size Analysis
* Profitability Ratios
* Liquidity Ratios
* Efficiency Ratios
* Credit Ratios

---

### 4. Cost of Capital (WACC)

The model calculates the **Weighted Average Cost of Capital (WACC)** using:

* Risk-Free Rate
* Market Risk Premium
* Beta Estimation
* Cost of Debt

---

### 5. Beta & Regression Analysis

A **regression analysis** is performed to estimate the company's **Beta** using historical **market return data**.

---

### 6. Valuation Methods

Multiple valuation approaches are used to estimate the company’s **intrinsic value**:

* Discounted Cash Flow (DCF) Valuation
* Dividend Discount Model (DDM)
* Free Cash Flow to Firm (FCFF) Valuation
* Relative Valuation (Comparable Company Analysis)

---

### 7. Industry & Competition Analysis

The project also includes:

* Industry financial comparison
* Competitor analysis
* Benchmarking financial performance

---
