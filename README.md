# Financial Modeling & Valuation – Mastercard Inc.

## Project Overview
This project analyzes Mastercard Inc. using financial modeling and valuation techniques. 
The model evaluates historical performance, forecasts financial statements, and estimates intrinsic value.

## Key Components
- 3 Statement Financial Model
- Financial Ratio Analysis
- Revenue and Expense Forecasting
- Cash Flow Analysis
- Industry Benchmarking
- Valuation Analysis

## Tools Used
- Microsoft Excel
- Financial Modeling Techniques
- Financial Statement Analysis

## Dataset
Financial data sourced from company annual reports and financial databases.US SEC EDGAR

## Key Insights
- Revenue growth trend analysis
- Profitability and efficiency evaluation
- Market risk analysis using Beta and regression
